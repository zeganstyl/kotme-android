##### Условные конструкции
**Конструкция if-else** принимает условие, и если условие в круглых скобках истинно,
то выполняется блок после if, если ложно - выполняется блок после else:
```kotlin
val a = 5
if (a == 5) {
    println("Переменная a равна 5")
} else {
    println("Переменная a не равна 5")
}
```
Если необходимо проверить несколько альтернативных вариантов, то можно добавить выражения else if:
```kotlin
val a = 4
if (a == 5) {
    println("Переменная a равна 5")
} else if (a == 4){
    println("Переменная a равна 4") 
} else {
    println("Переменная a имеет другое значение")
}
```
Синтаксис Котлина позволяет присваивать переменной условное выражение,
что делает код более лаконичным.
Использование ветки else в таком случае является обязательным.
Например, найдем максимальное из двух чисел:
```kotlin
val a = 15
val b = 25
val c = if (a > b) a else b
println(c)  // 25
```
**Конструкция When** проверяет значение некоторого объекта и в зависимости от его значения
выполняет тот или иной код. Конструкция when аналогична конструкции switch в других языках.
```kotlin
val a = 10
when (a) {
    5 -> {
        println("===")
        println("a = 5")
    }
    10 -> println("a = 10")
    15 -> println("a = 15")
    else -> println("Переменная a имеет неопределенное значение")
}
```
В скобках, после when, записываем переменную, которая будет выступать в роли "переключателя".
Затем, в фигурных скобках, прописываем значения, которые может принимать эта переменная.
А для каждого значения, через стрелку пишем, код, который выполнится, если перемнная примет указанное значение.
Если, для значения, требуется вписать несколько операций, пишем их в фигурных скобках.

Есть еще одна конструкция when.
```kotlin
val a = 13
when {
    a > 10 -> println("переменная a больше 10")
    a < 10 -> println("переменная a меньше 10")
    else -> println("переменная a равна 10")
}
```
В данной конструкций, вместо переменной и её значений, мы вписываем условия,
при которых будут срабатывать операции. Принцип работы этой конструкции аналогичен записи
`if (a > 10) { ... } else if (a < 10) { ... } else { ... }`

#### Циклы
Циклы представляют вид управляющих конструкций, которые позволяют
повторять выполнение группы операций некоторое число раз.

Самый известный цикл, это цикл **for**. С помощью него можно указать начально,
конец и шаг наращивания переменной-счетчика, например:
Его синтаксис выглядит следующим образом:
```kotlin
for (i in 0 until 10) {
    println(i)
}
```
Выше приведнный код печатает цифры от 0 до 9.
С помощью такой конструкции можно, например, пройти по всем элементам массива.

Есть также альтернатвные записи цикла for. Например с опратором **in**.
Оператор in и !in проверяют задает диапазон перебираемых значений:
```kotlin
for(i in 1..10){
    println(i)
} 
```
В данном случае цикл пройдет по значениям от 1 до 10. 

**Цикл While** является наиболее универсальным циклом,
но его конструкция немного больше чем у for.
Главное отличие - в круглых скобках записываем условие.
И пока выполняется условие, цикл будет повторяться.
```kotlin
var i = 0
while (i < 5){
    println(i)
    i += 1
}
```
Данный код выведет цифры от 0 до 5.

Kotlin также поддерживает операторы **break** и **continue** в циклах.
Они используются, если необходимо выйти из цикла при достижении какого-то значения:
```kotlin
for(i in 1..10){
    println(i)
    // прекращаем перебор при достижении значения 3
    if (i == 3)
        break
}
```
Ключевое слово **continue** позволяет пропустить текущий шаг.
То есть, как только достигаем continue, происходит переход к следующем шагу,
минуя все остальные операции:
```kotlin
for(i in 1..10){
    if (i == 5) {
        print("abc")
        continue
    }
    print(i * i)
}
//Выводит   1 	4 	9 	16 	abc	36 	49 	64 	81 	100 
```

##### Лямбды
Лямбда – функция, которую можно присвоить в переменную и передавать в другие функции,
в качестве аргументов.
```kotlin
var f: (a: Int, b: Int) -> Int = { a, b -> a + b }
println(f(2, 5))
```
Тип перемнной для лямбды записывается, начиная с круглых скобок, где мы указываем параметры
нашей лямбда-функции. А далее, после стрелки, записыаем тип возращаемого значения.

В данном примере мы присвоили лямбду перемнной. Тело лямбды записано в фигурных скобках.
В теле мы должны вначале указать параметры, а затем, через стрелку, операции, выполняемы,
при вызове этой лямбды. Для заметки, имена параметров можно укзывать любые,
главное сохранить их порядок.

Вызывается лямбда, прямо как обычная функция. Указываем имя переменной и далее, в круглых скобках,
посылаем аргументы. Лямбду также вызвать с помощью слова **invoke**. Например `f.invoke(2, 5)`

##### Строковые шаблоны

Строки могут содержать шаблонные выражения, т.е. участки кода, которые выполняются, а полученный результат встраивается в строку. Так, с помощью символа `$` мы можем вводить в строку значения различных переменных:
```kotlin
fun main() {
    val name: String = "Tom"
    println("Hello, $name! How are you?")
}
//Выводит: Hello, Tom! How are you?
```
Строковые шаблоны часто используются при вызове функций println() и print(). Однако строковый шаблон с ними никак не связан. Это просто строка, которая как и всякая другая строка может быть присвоена переменной:
```kotlin
fun main() {
    val name = "Tom"
   	val hello = "Hello, $name! How are you?"
	println(hello)
}
```
